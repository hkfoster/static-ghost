Mongo in San Francisco

December 1st, 2011

If you missed us in Seattle, most the MongoHQ team will be in San Francisco for the largest MongoDB conference ever. Seven tracks – one day. Seriously huge. So stop by our booth and say hello, and you’ll get a special prize. A seriously special prize.

The MongoHQ github repo is getting filled up with lots of brand new code. We recently started a Java driver code-named Astérix, for the still secret MongoHQ API. This is not an official announcement, so don’t tell your friends just yet. But if you like Java and love MongoHQ (that’d be us… you know, the MongoDB hosting company) we certainly won’t stop you from spying the source code. It’ll end up in a Maven repository soon enough…

(If Seattle got a video, so should SF. My initial choice–We Built This City–was vetoed.)
