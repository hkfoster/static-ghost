MongoDB Elastic Deployments now available in Oregon

May 16th, 2014

We have finally completed Oregon Trail (been trying since 5th grade). We’re celebrating by launching Elastic Deployments optimized for use in the AWS us-west-2 datacenter. If you have a production, MongoDB backed application you want to run on the west coast, we’ll now handle the database for you.

Elastic Deployments give you the best of MongoDB hosting — fully replicated databases running on fast SSDs with access to huge amounts of RAM. You only pay for what you use, at a flat monthly fee of $18/GB. You also get:

  • Automatic daily backups, or run them On DemandIf you are already a MongoHQ customer, you can provision a new Elastic Deployment with the database creation form. Make sure you choose “AWS us-west-2 (Oregon)” in the region selector.

New to MongoHQ? We have a special signup link for Elastic Deployments in Oregon — create an account and get a database in us-west-2. It takes less than a minute to be off and running!
