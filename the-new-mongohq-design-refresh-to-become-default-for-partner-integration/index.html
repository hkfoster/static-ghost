The New MongoHQ Design Refresh to Become Default for Partner Integration

November 19th, 2012

Last month, we rolled out the new MongoHQ (The New MongoHQ – a Delightful Way to Manage Your Datababase). Now, we are enabling the same UI to our partners: Heroku, Cloudbees, AppFog, and AppHarbor, Nodejitsu, and EngineYard. With this release, our clients through Partners can access all of the same great features accessible directly with MongoHQ.

When using Partner integration with MongoHQ, to access the new site today, click on “Launch new MongoHQ”. On Friday, we will enable the New MongoHQ as the default view for all Partner interactions.
